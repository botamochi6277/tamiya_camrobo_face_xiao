# TAMIYA CAMROBO FACE XIAO

![front](https://user-images.githubusercontent.com/14128408/90113329-25f20780-dd8c-11ea-97de-271f3284443d.jpeg)
![back](https://user-images.githubusercontent.com/14128408/90113333-27233480-dd8c-11ea-8e0d-c0e0aaf72132.jpeg)

An extension board for the face of [TAMIYA Cam-Program Robot](https://www.tamiya.com/english/products/70227/index.htm). 
This board has [Seeeduino XIAO](https://www.seeedstudio.com/Seeeduino-XIAO-Arduino-Microcontroller-SAMD21-Cortex-M0+-p-4426.html), RGB LEDs and a piezo speaker. 

- LED: Neopixel, [WS2812B](http://akizukidenshi.com/catalog/g/gI-07915/)
- Speaker: [muRata SMD piezo Speaker](http://akizukidenshi.com/catalog/g/gP-05723/)
